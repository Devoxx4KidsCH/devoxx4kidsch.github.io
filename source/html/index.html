Balises

Une balise a un nom (p, html, head, title, h1, a) et s'ecrit entre les signes < et >. Les balises s'écrivent par paire pour agir sur ce qu'elles encadrent.

exemple :

<h1>Un peu de texte</h1>

Création d'une page :

Pour créer une page il faut commencer par la balise html :
<html></html>

Ensuite on ajoute un debut de document qui contiendra le titre et un corps de document avec les balises suivantes :

<html>
<head>
<meta charset="UTF-8">
<title> Mon titre </title>
</head>
<body>

</body>
</html>

A partir de maintenant, pour afficher quelque chose à l'écran, les balises doivent être placés dans le body

La balise image

Pour afficher une image il faut utiliser la balise img de la façon suivante :

<img src="URL" />
Pour changer la taille de l'image : <img style="width:500px;" src="http://urlDeLimage" />

Les titres

<h1>Gros titre</h1>
<h2>Moins gros titre</h2>

Le texte

Vous pouvez écrire du texte dans une balise <p> ici mon texte </p>

Changer la couleur

Pour changer la couleur du texte :

<p style="color:red;"></p>

Changer la couleur de fond de la page

<body style="background:green;">

Fond de page en dégradé

<body style="background-image:linear-gradient(white, black);">

Ajouter une favicon

mettre dans le head :

<link rel="icon" type="image/png" href="favicon.png" />

Les liens

<a href="URL">Lien</a>
